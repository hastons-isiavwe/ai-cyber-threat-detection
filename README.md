# AI Cyber Threat Detection System

This project presents a machine learning-based intrusion detection system designed to identify cyber threats in network traffic data. It applies multiple classification algorithms to detect malicious activities and evaluate their effectiveness in cybersecurity applications.

---

## 🚀 Key Features

- Multi-model intrusion detection system (IDS)
- Detection of malicious and normal network traffic patterns
- Implementation of machine learning algorithms:
  - Logistic Regression
  - Random Forest
  - Gradient Boosting
  - Support Vector Machine (SVM)
- Performance evaluation using:
  - Confusion Matrix
  - ROC Curves
  - Accuracy and precision metrics
- Comparative analysis to determine the best-performing model

---

## 🧠 Methodology

- Preprocessed network traffic datasets
- Applied feature selection and transformation
- Trained multiple supervised learning models
- Evaluated performance across models
- Visualized results using plots and performance charts

---

## 📊 Results

The project compares multiple models and identifies the most effective algorithm for detecting cyber threats based on performance metrics.

---

## ⚙️ Tech Stack

- Python  
- Scikit-learn  
- NumPy  
- Matplotlib  

---

## 🔍 Key Insights

- Ensemble models (e.g., Random Forest, Gradient Boosting) show strong performance  
- Proper feature selection significantly improves detection accuracy  
- Machine learning can effectively detect anomalies in network traffic
